# Graph Algorithm Visualizer

Interactive step-by-step visualizer for graph algorithms built with React.js

## Algorithms
- Dijkstra (Single Source Shortest Path)
- Bellman-Ford (handles negative weights + cycle detection)
- Prim's MST
- Kruskal's MST (Union-Find with path compression)

## Features
- Drag nodes, add/delete edges
- Step-by-step playback with speed control
- Live distance table
- Negative cycle detection

## Tech
React.js · SVG Canvas · CSS Variables · Vite
